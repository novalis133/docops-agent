# Security Policy Update 2026

## Password Requirements

All employees must follow these updated password guidelines:

- Passwords must be **minimum 16 characters** long
- Include at least one uppercase, lowercase, number, and symbol
- Password rotation required every **45 days**
- Cannot reuse the last 10 passwords

## Multi-Factor Authentication

MFA is now **mandatory** for all systems:
- Use company-approved authenticator apps
- Hardware tokens available upon request
- SMS-based MFA is no longer permitted

## Remote Access

When working remotely:
- VPN must be connected at all times
- Only company-issued devices allowed
- Public WiFi usage prohibited without VPN

## Data Classification

All documents must be classified as:
- **Confidential** - Internal use only
- **Restricted** - Need-to-know basis
- **Public** - Can be shared externally

## Effective Date

This policy is effective **February 1, 2026** and supersedes all previous security policies.
